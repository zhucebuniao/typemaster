原理

怎么做到的？

5 步，把像素变成字符

1

加载图片

用 Canvas API 读取图片像素数据，拿到每个像素的 RGBA 值。

2

缩放采样

把图片缩放到目标宽度（比如 120 字符），每个采样点对应一个字符。

3

计算亮度

每个像素转灰度：0.299R + 0.587G + 0.114B，得到 0-255 的亮度值。

4

映射字符

亮度值映射到字符集索引。暗→稀疏字符，亮→密集字符。

5

渲染输出

逐行拼接字符，用 Canvas 绘制带颜色的像素，或输出纯文本。

// 核心映射逻辑
const chars = " .:-=+*#%@";
const idx = Math.floor(brightness / 255 * (chars.length - 1));
return chars[idx];
