原理

怎么做到的？

浏览器里用 Canvas + Video API 实现实时转换

1

加载视频

用 HTML5 Video 元素加载视频，获取逐帧画面。

2

采样缩放

每帧绘制到小 Canvas，按目标宽度缩放，计算亮度。

3

映射字符

亮度值映射到字符集，可选不同调色板上色。

4

实时渲染

requestAnimationFrame 循环，逐帧渲染到输出 Canvas。

命令行

Python 版本

更强大的离线转换工具

# 安装依赖
pip install Pillow numpy

# 基本用法
python ascii-video-converter.py input.mp4

# 自定义参数
python ascii-video-converter.py input.mp4 -o output.mp4 -w 160 -c dense -p amber

# 参数说明
-w, --width     ASCII 宽度（默认 120）
-c, --charset   字符集（standard/dense/blocks/ascii/binary/dots）
-p, --palette   调色板（green/amber/blue/white/original）
-f, --fps       帧率（默认 24）
