原理

怎么做到的？

Python + FFmpeg + PIL = 视频转 ASCII

1

读取视频

用 FFmpeg 逐帧解码视频，获取每帧的 RGB 像素数据。

2

缩放采样

把每帧缩放到目标宽度（比如 120 字符），计算每个采样点的亮度。

3

映射字符

亮度值映射到字符集。暗→稀疏字符，亮→密集字符。

4

渲染输出

用 PIL 把字符画渲染成图像，再用 FFmpeg 编码成视频。

使用

命令行工具

一行命令，把视频变成 ASCII 风格

# 基本用法
python ascii-video-converter.py input.mp4

# 自定义参数
python ascii-video-converter.py input.mp4 -o output.mp4 -w 160 -c dense -p amber

# 参数说明
-w, --width     ASCII 宽度（默认 120）
-c, --charset   字符集（standard/dense/blocks/ascii/binary/dots）
-p, --palette   调色板（green/amber/blue/white/original）
-f, --fps       帧率（默认 24）
