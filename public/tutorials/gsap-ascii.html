原理

怎么做到的？

GSAP 驱动数据，数据驱动字符

1

创建字符网格

用一个二维数组存储每个字符的位置、颜色、透明度等属性。

2

定义动画目标

每个字符的目标状态：位移、缩放、颜色变化、透明度。

3

GSAP 补间

用 gsap.to() 对每个字符创建补间动画，设置缓动函数和延迟。

4

渲染更新

每帧读取字符属性，重新渲染到 DOM 或 Canvas。

// GSAP 驱动字符动画
const chars = document.querySelectorAll('.char');

// 波浪动画
gsap.to(chars, {
  y: (i) => Math.sin(i * 0.3) * 50,
  color: (i) => `hsl(${i * 10}, 80%, 60%)`,
  duration: 1.5,
  ease: "elastic.out",
  stagger: 0.02
});
