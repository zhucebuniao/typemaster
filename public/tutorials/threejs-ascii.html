原理

怎么做到的？

Three.js + Canvas + 字符映射 = 3D ASCII

1

渲染 3D 场景

用 Three.js 创建场景、相机、灯光、几何体，渲染到离屏 Canvas。

2

读取像素

从 Canvas 读取每帧的像素数据（getImageData），获取 RGB 值。

3

计算亮度

每个像素转灰度：0.299R + 0.587G + 0.114B，得到亮度值。

4

映射字符

亮度映射到字符集，暗→稀疏，亮→密集。保留原始颜色。

// 读取 3D 渲染结果
const imageData = ctx3d.getImageData(0, 0, w, h);

// 遍历像素
for (let y = 0; y < h; y += cellH) {
  for (let x = 0; x < w; x += cellW) {
    const i = (y * w + x) * 4;
    const r = data[i], g = data[i+1], b = data[i+2];
    const lum = (0.299*r + 0.587*g + 0.114*b) / 255;
    const char = chars[Math.floor(lum * chars.length)];
    // 用原始颜色绘制字符
    ctxAscii.fillText(char, x, y, {color: `rgb(${r},${g},${b})`});
  }
}
渲染模式

不同的降维风格

同一种 3D 场景，不同的字符渲染方式

标准

░▒▓█ 字符集，保留原始颜色，最接近原图。

密集

▁▂▃▄▅▆▇█ 字符集，更平滑的渐变效果。

经典

@%#*+=-:. 字符集，纯文本风格，适合终端。

二进制

01 字符集，黑客帝国风格，极致简约。
