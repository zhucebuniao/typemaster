第一章

场景三部曲

Scene + Camera + Renderer——Three.js 的基石

几乎每个 Three.js 程序都从这三行开始
最小示例JavaScript
const scene = new THREE.Scene()
const camera = new THREE.PerspectiveCamera(75, w/h, 0.1, 1000)
const renderer = new THREE.WebGLRenderer({ antialias: true })
Scene
场景容器
放所有物体
Camera
视角窗口
决定看到什么
Renderer
渲染引擎
画到 canvas
关键参数说明
// PerspectiveCamera(fov, aspect, near, far)
// fov: 视野角度（度），默认 45-75
// near/far: 近远裁剪面，只渲染之间的物体
//
// WebGLRenderer({ antialias: true }) 抗锯齿
// renderer.setSize(w, h) 设置渲染尺寸
// renderer.render(scene, camera) 渲染单帧

MeshStandardMaterial

PBR 材质，最常用
roughness + metalness 控制

MeshPhongMaterial

经典光照模型
shininess 控制高光

MeshLambertMaterial

漫反射光照
性能好，效果简单

MeshBasicMaterial

不受光照影响
纯色显示，用于调试

完整场景代码（核心片段）JavaScript
// 1. 搭建场景三件套
const scene = new THREE.Scene()
const camera = new THREE.PerspectiveCamera(45, w/h, 0.1, 100)
camera.position.set(0, 3, 8)
const renderer = new THREE.WebGLRenderer({ antialias: true })

// 2. 加入物体、光照、地面
createTorusKnot(scene)
addLights(scene)
addGrid(scene)

// 3. 启动动画循环
function animate() {
  requestAnimationFrame(animate)
  mesh.rotation.x += 0.005
  mesh.rotation.y += 0.01
  renderer.render(scene, camera)
}
