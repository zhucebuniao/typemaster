核心 Tween

gsap.to() · gsap.from() · gsap.fromTo()——三种基本形态

当前方法： gsap.to()
→
示例代码JavaScript
// 将元素从当前位置移动到 x:200
gsap.to(".box", {
  x: 200,
  duration: 1,
  ease: "power2.out"
})

gsap.to()

从当前状态 → 目标状态

gsap.to(el, { x: 100 })

gsap.from()

从起始状态 → 当前状态

gsap.from(el, { x: 100 })

gsap.fromTo()

明确指定起止状态

gsap.fromTo(el, { x: 0 }, { x: 100 })
